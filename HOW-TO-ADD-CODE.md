# How to add your code manually

## Folder structure

```
gds-site-manual/
  index.html       ← Main page (header, hero, sections, footer)
  css/
    style.css      ← All your styles
  js/
    main.js        ← All your scripts
  HOW-TO-ADD-CODE.md  ← This file
```

## Option 1: You paste in Cursor

1. Open this folder in Cursor: **File → Open Folder** → select **gds-site-manual**.
2. Open **index.html**. Replace the comment blocks with your code for each section (header, hero, etc.).
3. Open **css/style.css**. Paste your layout and section CSS.
4. Open **js/main.js**. Paste your JavaScript.
5. Save each file (Ctrl+S).

## Option 2: Send code here and we add it

Tell me which file each piece belongs to, for example:

- "This is the header" → I'll put it in index.html in the header section.
- "This is the CSS for the hero" → I'll add it to css/style.css.
- "This is the contact form section" → I'll add it to index.html in the sections area.

Paste the code in your message and say the section name (e.g. "header", "hero", "footer", "styles", "script").

## Multiple pages

If you have separate pages (e.g. contact.html, services.html), tell me and I'll create those files. You can then paste the code for each page.

## After adding code

- Test locally: open **index.html** in a browser, or run a simple server.
- When ready, push this folder to GitHub and deploy to Vercel (same steps as before).
